# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added
- **Core Engine**: FPFH-based feature detection and template matching
- **CFD Integration**: Automated generation of `snappyHexMeshDict`
- **MRF Support**: Moving Reference Frame zone generation for rotating components
- **CLI**: `automesh` command-line tool for batch processing
- **SDK**: Python API (`meshmind.sdk.mesher`) for custom workflows
- **Docs**: Comprehensive documentation and usage guides
- **Distribution**: 
  - Standalone binary builds for macOS/Linux
  - `pip` installation support

### Security
- No known vulnerabilities
